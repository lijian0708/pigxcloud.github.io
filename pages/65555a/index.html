Ubuntu 开发环境变量配置

# Ubuntu 18.04 开发环境变量配置

# JDK

# 下载安装包

访问JDK官方下载页面 (opens new window) ，下载Linux x64 Compressed Archive版本安装包，本教程使用的版本为：jdk-8u261-linux-x64.tar.gz。

勾选同意协议，点击下载

首次使用登录Oracle账号，无果没有账号免费注册一个即可。登录成功会自动开始下载。

# 上传并解压

$ sudo mkdir /usr/local/java
$ sudo tar -zxvf jdk-8u261-linux-x64.tar.gz -C /usr/local/java

# 配置环境变量

$ sudo vi /etc/profile

在文件末尾插入如下内容

# jdk
export JAVA_HOME=/usr/local/java/jdk1.8.0_261
export JRE_HOME=${JAVA_HOME}/jre
export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
export PATH=${JAVA_HOME}/bin:$PATH

# 执行source /etc/profile

$ source /etc/profile

# 检查jdk是否生效java -version

heyuqiang@heyuqiang:/usr/local/java/jdk1.8.0_261$ java -version 
java version "1.8.0_261"
Java(TM) SE Runtime Environment (build 1.8.0_261-b12)
Java HotSpot(TM) 64-Bit Server VM (build 25.261-b12, mixed mode)

# Maven

# 下载安装包

访问Maven官方下载页面 (opens new window) ，下载Binary tar.gz archive版本安装包，本教程使用的版本为：apache-maven-3.6.3-bin.tar.gz。

# 上传并解压

$ sudo mkdir /usr/local/maven
$ sudo tar -zxvf apache-maven-3.6.3-bin.tar.gz -C /usr/local/maven

# 配置环境变量

$ sudo vi /etc/profile

在文件末尾插入如下内容

# maven
export NODEJS_HOME=/usr/local/maven/apache-maven-3.6.3
export PATH=$NODEJS_HOME/bin:$PATH

# 执行source /etc/profile

$ source /etc/profile

# 检查maven是否生效mvn -v

heyuqiang@heyuqiang:/usr/local/maven/apache-maven-3.6.3$ mvn -v

# NodeJs

# 离线安装 推荐

推荐大家使用离线安装方式，首先根据自己的需要先从官网下载需要的releases版本安装包 (opens new window)，我这里下载安装的node版本为v10.22.0，解压配置好环境变量即可。

# 下载安装包

使用wget命令下载二进制安装包node-v10.22.0-linux-x64.tar.gz

$ wget https://nodejs.org/dist/latest-v10.x/node-v10.22.0-linux-x64.tar.gz

# 上传并解压

$ sudo mkdir /usr/local/lib/nodejs
$ sudo tar -zxvf node-v10.22.0-linux-x64.tar.gz -C /usr/local/lib/nodejs/

# 配置环境变量

$ sudo vi /etc/profile

在文档末尾追加如下内容

# nodejs
export NODEJS_HOME=/usr/local/lib/nodejs/node-v10.22.0-linux-x64
export PATH=$NODEJS_HOME/bin:$PATH

# 执行source /etc/profile

$ source /etc/profile

# 执行node -v和npm -v命令检查软件版本

$ node -v
v10.22.0
$ npm -v 
6.14.6

# 在线安装

提示

默认使用 apt 安装的 node 和 npm 版本可能比较老，需要更新一下对应的版本

# 使用 apt 安装 node

sudo apt install nodejs

# 使用 apt 安装 npm

sudo apt install npm

$ nodejs -v
v8.10.0
$ npm -v
3.5.2

# npm 更新

sudo npm i -g npm

重新打开终端，查看 npm 版本，如下版本已经是最新

$ npm -v
6.7.0

# node 更新

先安装 n 模块

sudo npm install -g n

$ sudo npm install -g n
/usr/local/bin/n -> /usr/local/lib/node_modules/n/bin/n
+ n@2.1.12
added 1 package from 4 contributors in 0.649s

执行 sudo n latest 命令更新 node

$ sudo n latest

     install : node-v11.8.0
       mkdir : /usr/local/n/versions/node/11.8.0
       fetch : https://nodejs.org/dist/v11.8.0/node-v11.8.0-linux-x64.tar.gz
   installed : v11.8.0

重新打开一个终端查看版本

$ node -v
v11.8.0

# 淘宝 NPM 镜像

使用说明 (opens new window)

# npm 安装yarn

# 一句命令搞定

npm install -g yarn --registry=https://registry.npm.taobao.org

# 再配置下源

yarn config set registry https://registry.npm.taobao.org -g
yarn config set sass_binary_site http://cdn.npm.taobao.org/dist/node-sass -g
