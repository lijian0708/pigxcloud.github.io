Nexus配置阿里云代理仓库

# Nexus配置阿里云代理仓库 推荐

# 概述

Nexus默认远程仓库为https://repo1.maven.org/maven2/，国内访问速度慢死，还常连不上。设置阿里云代理仓库可加速访问速度。

# 添加阿里云代理仓库

创建仓库

选择maven2 (proxy)

仓库设置

  • Name: maven-aliyun
  • Remote storage: https://maven.aliyun.com/repository/public/

阿里云已经更新了仓库地址，图片上为旧版本

其他默认，翻到最下面，点create repository保存

仓库状态显示Online - Remote Available表示配置成功

# 配置public-repository

点击maven-aliyun，再点击>按钮

设置Nexus获取依赖的优先顺序。如下图，优先去阿里云的Maven

至此，Nexus配置阿里云代理仓库完成。
